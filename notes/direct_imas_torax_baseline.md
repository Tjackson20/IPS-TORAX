# Direct IMAS TORAX Baseline

## Purpose

This baseline proves that TORAX can run directly from the IMAS HDF5 files in
`data/` without using the manual IMAS-to-TORAX mapper as the main workflow.

## Input Data

- IMAS URI: `imas:hdf5?path=data`
- IDSs loaded:
  - `equilibrium`
  - `core_profiles`
  - `core_sources`
- Source files:
  - `data/equilibrium.h5`
  - `data/core_profiles.h5`
  - `data/core_sources.h5`
  - `data/master.h5`
- Input DD version: `3.42.0`
- TORAX expected DD version: `4.0.0`
- Required load option: `explicit_convert=True`

## Configuration

- Config file: `scripts/imas_torax_config.py`
- Geometry: direct IMAS geometry from `equilibrium`
- Radial grid: `n_rho = 25`
- Transport model: `constant`
- Solver: `linear`
- Time step calculator: `fixed`
- `t_initial = 0.0`
- `t_final = 0.1`
- `fixed_dt = 0.1`
- Evolved equations:
  - current
  - ion heat
  - electron heat
  - density

## IMAS Functions Used

- `loader.load_imas_data(...)`
- `core_profiles.profile_conditions_from_IMAS(...)`
- `core_profiles.plasma_composition_from_IMAS(...)`
- `core_sources.sources_from_IMAS(...)`

## Source Handling

TORAX maps these IMAS `core_sources` entries into runnable TORAX sources:

- `ecrh`
- `fusion`
- `ohmic`
- `ei_exchange`

The direct IMAS baseline excludes `icrh` because this TORAX installation tries
to load a local TORIC surrogate model file that is not present:

- `~/toric_surrogate/TORIC_MLP_v1/toricnn.json`

Several IMAS source names are skipped by TORAX because there is no built-in
source-name mapping for them in the current TORAX loader.

## Verification

Plots are generated by:

```bash
/home/teliyah/miniforge3/envs/TORAX/bin/python scripts/plot_imas_torax_profiles.py
```

Plot directory:

```text
plots/imas_vs_torax/
```

The plots compare:

- IMAS input
- TORAX initial state
- TORAX final state

The key check is that TORAX initial values match IMAS closely, proving the
simulation starts from the IMAS files. The TORAX final state is expected to be
different because the simulation evolves the plasma.

Representative central values from the current baseline:

| Quantity | IMAS | TORAX initial | TORAX final |
| --- | ---: | ---: | ---: |
| `T_e` axis, keV | 31.949 | 31.9169 | 29.5186 |
| `T_i` axis, keV | 23.6679 | 23.6535 | 21.5374 |
| `n_e` axis, m^-3 | 1.16706e20 | 1.16707e20 | 1.32161e20 |
| `Ip`, A | 1.5e7 | 1.5e7 | 1.48898e7 |
