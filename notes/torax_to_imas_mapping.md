# TORAX to IMAS Mapping

## core_profiles

T_e -> electron temperature
T_i -> ion temperature
n_e -> electron density
pressure_total -> plasma pressure
psi -> poloidal flux

## equilibrium

q -> safety factor profile
j_total -> total current profile
Ip -> plasma current

## core_transport

chi_turb_i -> ion turbulent diffusivity
chi_turb_e -> electron turbulent diffusivity
chi_neo_i -> ion neoclassical diffusivity
chi_neo_e -> electron neoclassical diffusivity

## core_sources

p_alpha_i -> fusion ion heating
p_alpha_e -> fusion electron heating
p_ohmic_e -> ohmic heating
j_parallel_bootstrap -> bootstrap current

## summary

tau_E -> energy confinement time
P_fusion -> fusion power
Q_fusion -> fusion gain
q95 -> edge safety factor
beta_N -> normalized beta
